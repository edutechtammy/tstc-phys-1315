PROMPT What is the standard unit of length in the SI system?

Answer

meter

Not what you got? This one is more of a memorization task. You will need to learn all the base units.

Length/distance: meter

Mass: gram

Volume: liter (remember that if you use the symbol to use uppercase for this one, L. This is because the lowercase l is so easy to confuse with a 1 when handwritten and in many typed fonts.
